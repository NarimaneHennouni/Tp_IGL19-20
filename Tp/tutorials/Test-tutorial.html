After cloning the repository from github and extracting the folder in emplacement you want

  • open the folder in a text editor
  • open the terminal and make sure that you are in the directory of the folder you extracted
  • split the terminal by using the split option in the top right corner of the terminal
  • in the left terminal run the application using the command "nmp run dev"
  • in the right one type the command "npm run test"
    • After a short moment you should see a geen label saying "passed" indicating that the test has passed
