Launching Tutorial

After cloning the repository from github and extracting the folder in emplacement you want

  • open the folder in a text editor
  • open the terminal and make sure that you are in the directory of the folder you extracted
  • type the commande "npm run dev" and wait for the app to start
    • Now you should have a page opened in the browser containing a table of the all students and their marks and averages
